# XwXJavve's Dev Kit

Personal development toolkit for Claude Code — integrated skills and MCP tools.

## Installation

```bash
cc --plugin-dir /path/to/dev-kit
```

Or copy to your project's `.claude-plugin/` directory.

## MCP Servers

### Tavily (Web Search)

[Tavily](https://tavily.com/) MCP provides web search capabilities.

**Setup:**

```bash
claude plugin configure dev-kit
```

Then enter your Tavily API key when prompted. Run `/mcp` to verify the server is connected.
